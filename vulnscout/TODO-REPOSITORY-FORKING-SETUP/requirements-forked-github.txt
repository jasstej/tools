# VulnScout Forked Tools - Installation from GitHub Organization
# Usage: pip install -r requirements-forked-github.txt
#
# Make sure you:
# 1. Are in the virtual environment: source ~/.vulnscout-env/bin/activate
# 2. Replace YOUR_ORG with your actual GitHub organization name
# 3. Have SSH key configured OR use HTTPS with personal token
#
# IMPORTANT: Replace "YOUR_ORG" with your GitHub organization name!
# Example: if your org is "my-security-tools":
#   git+https://github.com/my-security-tools/commix.git

# Using HTTPS (requires GitHub token for private repos)
# Uncomment and update YOUR_ORG to use these URLs:
#
# git+https://github.com/YOUR_ORG/commix.git
# git+https://github.com/YOUR_ORG/ghauri.git
# git+https://github.com/YOUR_ORG/wafw00f.git
# git+https://github.com/YOUR_ORG/LinkFinder.git
# git+https://github.com/YOUR_ORG/paramspider.git
# git+https://github.com/YOUR_ORG/arjun.git
# git+https://github.com/YOUR_ORG/SubDomainizer.git
# git+https://github.com/YOUR_ORG/jwt-tool.git
# git+https://github.com/YOUR_ORG/dotdotpwn.git
# git+https://github.com/YOUR_ORG/anew.git

# Using SSH (requires SSH key configured)
# Uncomment and update YOUR_ORG to use these URLs:
#
# git+ssh://git@github.com/YOUR_ORG/commix.git
# git+ssh://git@github.com/YOUR_ORG/ghauri.git
# git+ssh://git@github.com/YOUR_ORG/wafw00f.git
# git+ssh://git@github.com/YOUR_ORG/LinkFinder.git
# git+ssh://git@github.com/YOUR_ORG/paramspider.git
# git+ssh://git@github.com/YOUR_ORG/arjun.git
# git+ssh://git@github.com/YOUR_ORG/SubDomainizer.git
# git+ssh://git@github.com/YOUR_ORG/jwt-tool.git
# git+ssh://git@github.com/YOUR_ORG/dotdotpwn.git
# git+ssh://git@github.com/YOUR_ORG/anew.git
